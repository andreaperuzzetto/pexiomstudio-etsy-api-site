PexiomStudio

AgentPeXI Seller Ops Assistant

Private seller operations workspace for one owner-operated Etsy shop. AgentPeXI helps prepare and review listing records, manual publish packs, pricing decisions, compliance checks, and analytics.

Internal tool Owner operated Official API only Manual-assisted publishing

Current API phase

Read-only mapping

  • Maps Etsy shop and listing records to internal records.
  • No listing creation, editing, or publishing in phase one.
  • No sales-data sync or email sending in phase one.

Operational Purpose

AgentPeXI supports internal seller operations for PexiomStudio: listing organization, product records, manual publish preparation, pricing review, compliance checks, and analytics review.

Access And Users

This application is used only by the shop owner/operator. It is not a buyer tool, public seller platform, or commercial marketplace app.

Publishing Model

Publishing remains manual-assisted. Any future write action would require explicit owner review and approval before it is performed.

Control Boundaries

The application is designed to stay conservative, auditable, and aligned with the approved Etsy API purpose.

  • OK Uses Etsy's official API only
  • OK Uses OAuth and minimal scopes
  • OK Does not scrape Etsy
  • OK Does not use browser automation
  • OK Does not send email
  • OK Does not publish hands-free
  • OK Does not train AI models on Etsy data
  • OK Keeps API credentials private

First Phase

The first API phase is limited to read-only shop and listing mapping so local AgentPeXI records can be connected to the owner's Etsy shop records.

Privacy

Data accessed through the Etsy API is used only for internal PexiomStudio shop operations. It is not sold, shared, used for advertising, or used to train AI models. API credentials and tokens are stored privately and are never published on this website.
